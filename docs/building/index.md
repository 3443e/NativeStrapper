# Building

This section covers building NativeStrapper from source.

Building is only necessary if you want to:
- Modify NativeStrapper
- Test unreleased changes
- Package your own builds
- Kill yourself

***You can also build through github actions! (https://github.com/3443e/NativeStrapper/actions)***

## Currently supported platforms

- [Linux (x86_64)](linux.md)
- [Windows (x64)](windows.md)
